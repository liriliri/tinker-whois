# tinker-whois 

A WHOIS query plugin for [TINKER](https://github.com/liriliri/tinker), supporting domain, IP address, and ASN lookups.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-whois/master/screenshot.png)

## Features

- **Domain, IP, ASN** queries
- **Fast response** (direct WHOIS TCP)
- **Modern UI**

## Installation

Download and install TINKER from `https://tinker.liriliri.io/`, then run `npm i -g tinker-whois`.

## Usage

1. Enter a domain, IP address, or ASN in the input box
2. Click the "Query" button or press Enter
3. View the WHOIS query results

## Example Queries

- `google.com` - Query Google domain information
- `github.com` - Query GitHub domain information
- `8.8.8.8` - Query Google DNS server information
- `AS15169` - Query Google's ASN information
